## 👋 Hi, I'm Giuliano

Data scientist and AI engineer with 5+ years of experience, mostly as a consultant for companies like Ambev, JBS, Leroy Merlin, and Engie. I work where data science and AI overlap: predictive modeling on one side, LLM systems (RAG, multi-agent, MCP) on the other. The problems I enjoy most usually sit between the two.

Most of what I learn ends up here. A few projects worth a look:

### 🌳 Model Tree
An interactive taxonomy of 365 predictive models, from linear regression to transformers to Bayesian hierarchical models, all in one navigable tree. Each node explains how a model differs from its siblings, its strengths and weaknesses, and when to use it (and when not to). Built with React 18 and D3.js, fully data-driven from a single JSON source, deployed static with zero backend.

### 🔎 IFRS Canoas RAG
A retrieval-augmented generation system for querying an institution's website in natural language. Full pipeline: crawling the pages, extracting text from HTML and PDFs, chunking, generating embeddings, indexing in Qdrant, retrieving context, and generating answers with Gemini.

### 🎮 League Bets
A machine learning system that predicts professional League of Legends match outcomes from real Riot API data. End-to-end: a scraping pipeline feeding roughly 29,000 matches, 14,000 players, and 2,300 teams, then classification and regression models for match winner and total kills, reaching 70% accuracy on held-out validation.
